# Daily Market Assistant
一个用于每日市场跟踪的 Python 自动化项目，主要功能包括：
- 生成A 股 ETF 海外 等标的交易量 KDJ MACD 等与量化因子分析；
- 自动获取每日投资语录；
- 估算基金当日收益表现；
- 生成分析图片与基金预估收益表；
- 通过邮件自动发送每日市场提醒；
- 支持 GitHub Actions 定时运行。
## 功能概览
### 1. 量化因子分析
项目会对配置好的市场标的生成日线、周线、月线分析，并输出对应图表。
支持的分析内容包括：
- ATR 等波动指标；
- 趋势与信号表；
- 午盘 / 收盘行情摘要。
### 2. 基金收益预估
`tools/get_top10_holdings.py` 支持输入一个或多个基金代码，估算基金当日涨跌幅，并生成汇总表格图片。
支持类型：
- 普通股票型基金；
- QDII 股票基金；
- ETF 联接基金；
- 指数联接基金；
- FOF 基金；
- 含 A 股、美股、港股持仓的混合基金。
估算逻辑：
- 普通基金：读取最近披露的前十大持仓；
- ETF 联接 / FOF：可使用代理 ETF 或指数基金进行估算；
- 若部分持仓无法获取行情，会自动剔除失败持仓，并将可获取行情的持仓重新归一化估算；
- 表格按今日预估涨跌幅从高到低排序；
- 输出限购金额、涨跌幅颜色、水印和备注。
### 3. 邮件自动发送
`main.py` 会整合：
- 每日语录；
- RSI 分析文字；
- 市场分析图片；
- 基金收益预估表格；
并通过邮件发送到指定邮箱。
